# Prodesk IT mission number 2 

This is a part of assignment from Prodesk IT internship.

## Expense Tracker

- Add and remove expenses dynamically
- Persistent data storage using Local Storage
- Real-time balance and expense calculation
- Doughnut chart visualization for spending vs remaining balance
- Currency conversion support using the Frankfurter API
- PDF export for expense reports
- Budget warning alerts when expenses exceed available balance
